List of Abbreviations
=====================


.. glossary::

   API
     Application Programming Interface.
     An interface between software components.

   ISP
     In System Programmer
     A tool that allows uploading new firmware to a device without requiring
     disassembly or other invasive actions.

   LSS
     Layer Setting Services.
     A CANopen protocol that allows the configuration of individual node ids
     and communication bitrates.

   REST
     Representational State Transfer.
     A paradigm for the design of :term:`API`\ s. The ``lxa-iobus-server`` provides
     a REST API on top of HTTP.
